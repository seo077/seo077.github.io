# JAVA

## ①

```java
class Solution {
    public long[] solution(int x, int n) {
        long[] answer = new long[n];
        long xlong = x;
        for(var i=0;i<n;i++){
            answer[i] = xlong + (xlong*i);
        }
        return answer;
    }
}

```

## ②

```java
class Solution {
    public long[] solution(int x, int n) {
        long[] answer = new long[n];
       answer[0] = (long)x;
        for(var i=1;i<n;i++){
            answer[i] += answer[i-1] + (long)x;
        }
        return answer;
    }
}

```

# JAVASCRIPT

## ①

```js
function solution(x, n) {
  var answer = [];
  for (var i = 0; i < n; i++) {
    answer.push(x + x * i);
  }
  return answer;
}
```

## ②

```js
function solution(x, n) {
  var answer = [];
  answer.push(x);
  for (var i = 1; i < n; i++) {
    answer[i] = answer[i - 1] + x;
  }
  return answer;
}
```
